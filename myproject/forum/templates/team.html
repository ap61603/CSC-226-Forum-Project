Our Weekly Updates

Step 1: Getting Started

We brainstormed ideas and decided to create a class forum with mini-games. We divided tasks and set up our file structure. Our goal is to create an interactive site where ourselves and our classmates will be able to access and create their own posts, play mini-games and more to come!.

Step 2: Forum Base

We built the basic HTML structures that we needed in order to create the mini-games and the forum, started on CSS styling to make the page look more professional, and finally we began planning the logic behind the mini-games. We know it will have to be kept simple.

Step 3: Finishing Game Development

We finished developing the two-player tic tac toe game along with the trivia questions by using HTML for the structure, JavaScript for the interactive gameplay and styling it with CSS to make it more appealing for the user. We also added an image icon to our website for a more complete look.

Step 4: Backend & Database Integration

We focused on enabling persistent storage for the forum posts by transitioning our project to a Django backend. After facing compatibility issues with Flask we successfully restructured the project by organizing HTML files into a 'templates' directory and moving CSS and JavaScript to a 'static' directory. We also configured the local database and updated the README with clear instructions so anyone can run the server locally.

Follow our journey - Ari, Sal, and Albert
